# Poker Solver

**A poker solver and analysis tool** written in Python and React, designed to help analyze player behavior, simulate hands, and test strategies using neural networks and MCTS (Monte Carlo Tree Search).  

---

## Features

- Exploitative poker solver that adapts to opponents' behavior
- Hand parsing and analysis tools
- Preflop and postflop simulation
- Neural network-based decision support
- Designed for cash game analysis
